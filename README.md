# PasswordTester-Py
A Python GUI application to test the time taken to crack your password 

# Requirements
* Python3
* PyQt5

# How to use
* Download zip - https://github.com/kannan95/PasswordTester-PyQt5
* Unzip the file
* cd ~/Downloads/PasswordTester-PyQt5-master
* python3 main.py
